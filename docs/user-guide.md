# 用户使用指南

## 1. 系统概述

预约系统是一个面向商户和用户的预约管理平台。商户可以创建预约任务，用户可以通过签名链接进行预约。

## 2. 用户角色

系统支持三种用户角色：

| 角色 | 权限说明 |
|------|----------|
| **普通用户 (USER)** | 注册账号、浏览服务、创建预约、查看预约记录 |
| **商户 (MERCHANT)** | 商户资料管理、服务项目管理、创建预约任务、查看预约统计 |
| **管理员 (ADMIN)** | 全平台统计查看、用户管理（后续扩展） |

## 3. 普通用户功能

### 3.1 注册与登录
- 访问系统注册页面，创建用户账号
- 使用用户名和密码登录系统

### 3.2 浏览服务
- 查看可用服务项目列表
- 查看服务详情（时长、价格、描述）

### 3.3 预约流程
- 通过商户分享的签名链接访问预约页面
- 选择预约日期和时间段
- 填写预约信息并提交
- 查看预约结果

### 3.4 我的预约
- 查看所有预约记录
- 取消待处理的预约

## 4. 商户用户功能

### 4.1 商户资料管理
- 维护商户基本信息（店铺名称、描述、联系方式、地址）
- 配置会话超时设置
- 设置通知偏好

### 4.2 服务项目管理
- 创建新的服务项目
- 编辑服务项目信息（名称、分类、时长、价格）
- 启用/禁用服务项目

### 4.3 预约任务管理
- 创建预约任务（选择服务、日期、总容量）
- 管理预约时段（添加、修改、删除时段）
- 查看预约列表和详情

### 4.4 统计分析
- 查看服务维度的预约统计
- 查看商户维度的经营数据

## 5. 管理员功能

### 5.1 平台统计
- 查看全平台用户数量
- 查看全平台预约数据
- 查看系统健康状态

## 6. 快速开始

### 6.1 启动系统

```bash
# 启动后端
./mvnw spring-boot:run

# 启动前端
cd frontend
npm install
npm run dev
```

### 6.2 访问系统

- 后端 API：http://localhost:8080
- 前端应用：http://localhost:5173
- API 文档：http://localhost:8080/swagger-ui.html

## 7. 常见问题

### 7.1 预约失败怎么办？
- 检查网络连接是否正常
- 确认选择的时段是否还有可用名额
- 查看是否有重复预约限制

### 7.2 商户如何生成预约链接？
- 登录商户账号
- 进入服务管理页面
- 选择需要生成链接的服务
- 系统将自动生成签名链接供分享
