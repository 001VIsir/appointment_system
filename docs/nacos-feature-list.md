## NACOS-001 - Nacos 客户端依赖引入

- **描述**: 在 pom.xml 中引入 Spring Cloud Alibaba Nacos 依赖
- **步骤**:
  1. 在 pom.xml 中添加 Spring Cloud Alibaba 依赖管理
  2. 添加 nacos-config 和 nacos-discovery 依赖
  3. 添加 Spring Cloud BOM 统一版本管理
- **状态**: 未通过
- **优先级**: 高

---

## NACOS-002 - Nacos 配置类创建

- **描述**: 创建 Nacos 相关配置文件，支持配置中心和注册中心
- **步骤**:
  1. 在 application.properties 中添加 Nacos 配置项
  2. 创建 bootstrap.properties 配置文件
  3. 配置命名空间和分组
- **状态**: 未通过
- **优先级**: 高

---

## NACOS-003 - 多环境配置支持

- **描述**: 支持 dev/test/prod 多环境配置切换
- **步骤**:
  1. 配置不同命名空间
  2. 创建各环境配置文件示例
  3. 通过 spring.profiles.active 切换环境
- **状态**: 未通过
- **优先级**: 中

---

## NACOS-004 - @RefreshScope 动态刷新

- **描述**: 实现配置动态刷新，无需重启应用
- **步骤**:
  1. 在配置类上添加 @RefreshScope 注解
  2. 使用 @Value 注入配置
  3. 添加配置变更监听日志
- **状态**: 未通过
- **优先级**: 高

---

## NACOS-005 - 本地配置兜底方案

- **描述**: Nacos 不可用时使用本地配置，保证应用可用性
- **步骤**:
  1. 保留 application.properties 本地配置
  2. 配置 shared-dataids 加载本地配置
  3. 设置本地配置优先级
- **状态**: 未通过
- **优先级**: 高
